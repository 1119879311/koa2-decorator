### 技术栈 ###
服务器：koa2+koa-router+mysql+jwt    
客户端：vue+vue-router+vuex+element-ui


### 服务端 node-serve ###

cd node-server   

安装依赖
npn install    

开发环境
npm run dev  (NODE_ENV=development)

编译测试
npm run start (NODE_ENV=production)

pm2 生产环境 (NODE_ENV=production)
npm run build
npm run pm2

### 客户端 node-client ###

cd node-client

开发环境
npm run serve

打包生产环境 
npm run build
